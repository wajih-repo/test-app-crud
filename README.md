# test-app-crud
Simple Crud Application to list, add, edit and delete entries from a table. It is also possible to perform filtering in real time to reduce the number of rows. 
The user can on click select a row, and modify it.    
# Used Technologies
* Springboot
* Spring Security
* MySQL
* Vaadin v8 for the UI
# URLS
* localhost:8080 for public area
* localhost:8080/main for main application
  * login = admin
  * password = 123
* localhost:8080/admin to display an Admin area
